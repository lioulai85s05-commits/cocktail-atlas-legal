Privacy Policy / 隐私政策

Last updated / 最后更新：2026-02-23

This app, Cocktail Atlas / 鸡尾酒配方, is offline-first and does not actively collect sensitive personal data by default.

本应用 Cocktail Atlas / 鸡尾酒配方 默认为本地优先，不主动收集个人敏感信息。

1. Data We Collect / 我们收集的数据

Default mode / 默认模式：

  • No personal identity data collected.
  • No ad tracking.
  • No behavior log upload.
  • 不收集个人身份信息
  • 不进行广告追踪
  • 不上传用户行为日志

Optional cloud sync mode (enabled by user) / 可选云同步模式（用户主动开启）：

  • Only user-defined User ID and favorite recipe IDs are uploaded.
  • Purpose: cross-device synchronization.
  • 仅上传用户自定义 User ID 与收藏配方 ID 列表。
  • 用途：跨设备同步。

2. Data Storage / 数据存储

  • Local favorites are stored on device (UserDefaults).
  • Cloud sync data is stored in the user-configured LeanCloud service.
  • 本地收藏数据保存在设备端（UserDefaults）。
  • 云同步数据存储于用户配置的 LeanCloud 服务。

3. Data Sharing / 数据共享

  • We do not sell user data to third parties.
  • If cloud sync is enabled, requests are sent directly from user device to LeanCloud.
  • 我们不会将用户数据出售给第三方。
  • 若用户启用 LeanCloud，同步请求由用户设备直接发送至 LeanCloud。

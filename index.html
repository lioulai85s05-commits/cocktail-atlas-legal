Privacy Policy / 隐私政策

Last updated / 最后更新：2026-02-23

1. Data We Collect / 我们收集的数据

Default mode: no personal identity data, no ad tracking, no behavior log upload.

默认模式：不收集个人身份信息、不进行广告追踪、不上传行为日志。

Optional cloud sync: only user-defined User ID and favorite recipe IDs for sync.

可选云同步：仅上传用户自定义 User ID 与收藏配方 ID，用于同步。

2. Data Storage / 数据存储

Favorites are stored locally via UserDefaults. Cloud data is stored in user-configured LeanCloud service.

收藏默认保存在本地（UserDefaults）；云同步数据存储于用户配置的 LeanCloud。
